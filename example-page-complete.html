Router Demo

This page demonstrates the client-side router integration. The sidebar is injected once and never reloads.

Click any sidebar link - notice the sidebar stays completely stable with no flicker!

How It Works

1. On first load, the sidebar is inserted into the DOM

2. When you click a nav link, the router fetches the new page HTML

3. Only the .main-wrapper content is swapped

4. The sidebar HTML is never touched = zero flicker
